# JK Flip-Flop using Verilog HDL

## Overview
This repository contains the Verilog HDL implementation of a JK Flip-Flop and its corresponding testbench.

## Features
- Synchronous JK Flip-Flop
- Functional Verification
- Simulation using Verilator
- Waveform analysis using GTKWave

## JK Flip-Flop Truth Table

| J | K | Q(next) |
|---|---|---------|
|0|0|No Change|
|0|1|Reset|
|1|0|Set|
|1|1|Toggle|

## Tools Used
- Verilog HDL
- Verilator
- GTKWave
- Git & GitHub
